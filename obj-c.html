Objective-C

The following materials discuss Objective-C, an object-oriented model of the C language. The materials here assume knowledge of basic programming fundamentals, and some knowledge of the C language (although we will cover concepts in C along the way). Objective-C is associated most strongly with iOS development. However, iOS development today and in the future is and most likely will be done in Swift. Despite this fact, legacy code will persist in the years to come, and that code is in Objective-C. A strong iOS developer should have a working knowledge of both Objective-C and Swift, and these materials aim to provide some basis for that knowledge.

Variables & Types

Objective-C is a compiled language. This means the code we write, called source code, must be first compiled before it is executed. To execute source code, Objective-C needs to store the relevant data to be used later. One way to store such data is through variables. Every variable in Objective-C must have (1) a name, and (2) a type.
